# Зарплатный проект

Для корректной работы необходим локальный сертификат. Т.к. используются шифрованные куки необходимо https соединение.
Для этого можно использовать mkcert. Подробнее [mkcert](https://github.com/FiloSottile/mkcert)

### Установка сертификатов:
```
$ mkcert.exe -install - установка root сертификата в хранилище
$ mkcert "*.pskb.ad" - генерация wildcard сертификатов для web-приложения
```
Переименовать файлы в cert.pem и key.pem соответственно и переместить в conf/cert
